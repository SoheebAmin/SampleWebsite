Backgrounds Explained

This background page comprises of 4 main components:

    The header: This is a modified header provided by W3 schools. Their stylesheet is linked in the HTML.
    The changing backgrounds: Each background has its own stylesheet updated by JS. All sheets have the name stylesheet + 1. The JS function adds one until the last sheet and cycles back to 0.
    The page text and change button: This is basic HTML text my own styling.
    This explanation pop-up: This is a button and modal provided by bootstrap, with a bit of customization for button position and color with my own CSS.

As with most pages on this sample site, the stylesheets are from libararies, with a self-made stylesheet to customize provided styles as desired.
