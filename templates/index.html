Homepage Explained

This homepage has 4 main components:

    The Nav bar: Basic nav bar styled by W3 schools. The actual HTML resides within application.py, factored out to be the same for all pages.
    The video background: This is stock footage of waves added as a video play on loop. It styled to be full screen, with additional CSS to handle scaling.
    The page text: This is basic HTML text with some CSS styling.
    This explanation pop-up: This is a button and modal provided by bootstrap, with a bit of customization for button position and color with my own CSS.

As with most pages on this sample site, the stylesheets are from libararies, with a self-made stylesheet to customize provided styles as desired. Besides just viewing it in your browser, the source code for this page can be found here. The code has comments to explain what each portion of code is doing.
